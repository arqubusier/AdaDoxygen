Options that is not listed below is used as usual by Doxygen

INPUT
	is modified with same behavior 

RECURSIVE
	is modified with same behavior 
	
FILE_PATTERNS
	The following must be added:
	*.ads \
	*.adb

EXTENSION_MAPPING 
	= adb=C++ ads=C++

STRIP_FROM_PATH
	is modified so that the tmp-directory is prepepended 
	
EXTRACT_ALL
	Only print c++ code that has a user-comment if set to NO

EXTRACT_PRIVATE
	Only print c++ code that is public if set to NO
