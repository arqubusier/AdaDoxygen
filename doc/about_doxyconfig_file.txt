Options that is not listed below is used as usual by Doxygen

INPUT
	is modified with same behavior 

RECURSIVE
	is modified with same behavior 
	
FILE_PATTERNS
	The following must be added:
	*.ads \
	*.adb

EXTENSION_MAPPING 
	= adb=C++ ads=C++

STRIP_FROM_PATH
	is modified so that the tmp-directory is prepepended. Only absolute path is supported.
	
EXTRACT_ALL
	If EXTRACT_ALL=NO, no system comment will be appended

EXTRACT_PRIVATE
	Only print c++ code that is public if set to NO
	
HIDE_UNDOC_CLASSES
	If HIDE_UNDOC_CLASSES=NO, undocumented packages will be extracted as well.
